# 小白云简历生成助手 - 专业版

## 概述

小白云简历生成助手是一个基于Vue 3框架的现代化AI辅助工具，专为求职简历生成和个人信息管理而设计。该智能体提供了一个直观的界面，允许用户进行对话、创建笔记、生成专业简历以及管理个人资料。支持响应式设计，可在手机端和电脑端完美适配。

## 主要功能

### 1. 智能对话
- **多主题交互**：能够回答各种问题，提供信息查询、计算帮助等
- **简历扩写**：根据用户导入的资料智能扩写简历内容
- **专业响应**：提供专业的求职建议和简历优化指导

### 2. 简历生成
- **多模板支持**：提供多种精美的简历模板
- **中英文切换**：支持中文和英文简历生成
- **多格式输出**：支持TXT、Word文档（.docx）和PDF格式导出
- **智能内容生成**：根据导入的资料自动生成专业简历内容

### 3. 个人资料管理
- **个人资料导入**：支持导入个人基本信息
- **教育经历导入**：支持广东白云学院各校区、学院、专业的选择
- **工作经验导入**：记录工作经历和工作描述
- **项目经历导入**：记录项目名称、时间、描述和使用技能
- **专业技能导入**：记录技能名称、熟练度和描述
- **网站管理**：可添加个人网站、社交媒体账号等链接
- **账号管理**：管理专业账号信息
- **收藏夹管理**：保存重要参考资料和链接

### 4. 笔记功能
- **快速记录**：提供便捷的笔记区域，方便用户记录重要信息
- **随时访问**：笔记内容可随时查看和编辑

### 5. 智能助手扩写
- **资料读取**：自动读取用户导入的所有个人资料
- **智能扩写**：基于导入资料智能扩写简历内容
- **内容预览**：提供简历预览功能
- **一键生成**：支持跳转到简历生成器页面

## 技术架构

### 前端技术
- **Vue 3**：现代化的前端框架，采用Composition API
- **Vite**：快速的构建工具，提供极速的开发体验
- **响应式设计**：使用CSS媒体查询，完美适配手机端和电脑端
- **Flex布局**：现代化的CSS布局方案
- **动画效果**：侧边栏滑入动画、消息过渡效果

### 核心依赖
- **docx**：用于生成Word文档（.docx格式）
- **html2pdf.js**：用于将HTML转换为PDF文档
- **localStorage**：本地存储用户导入的资料

## 使用指南

### 启动应用
1. 确保已安装Node.js
2. 在项目根目录运行 `npm install` 安装依赖
3. 运行 `npm run dev` 启动开发服务器
4. 访问浏览器显示的本地地址（通常是 http://localhost:5173/）

### 开始对话
**方式1：**
1. 在欢迎页面点击"开始对话"按钮
2. 在输入框中输入问题或请求
3. 点击"发送"按钮或按Enter键提交
4. 智能体会根据您的请求提供相应回复

**方式2：**
1. 点击左侧侧边栏的"对话"导航项
2. 聊天界面会自动打开

### 导入资料
1. 点击左侧侧边栏的"导入资料"下的相应选项
2. 或点击"导入网站"、"导入账号"、"导入收藏夹"按钮
3. 填写相关信息
4. 点击"保存"按钮保存到本地存储

**支持的导入类型：**
- 个人资料
- 教育经历
- 工作经验
- 项目经历（新增）
- 专业技能（新增）
- 网站
- 账号
- 收藏夹

### 简历生成
1. 点击左侧"简历"菜单项或在对话中请求生成简历
2. 在简历生成器页面查看和编辑简历
3. 选择不同的模板
4. 切换中英文
5. 下载为TXT、Word或PDF格式

### 智能扩写简历
1. 先导入您的个人资料、教育经历、工作经验等信息
2. 在聊天界面输入"请帮我扩写简历"或类似请求
3. 智能助手会根据您导入的资料智能扩写简历内容
4. 您可以将扩写的内容复制到简历生成器中
